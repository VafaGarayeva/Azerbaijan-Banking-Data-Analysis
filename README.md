# Azerbaijan-Banking-Data-Analysis
Strategic analysis of digital banking and NFC adoption potential in Azerbaijan using SQL and Tableau.

This project analyzes the correlation between Baku Metro passenger flow and national banking trends to identify high-potential zones for NFC payments.

## 🛠️ Tools Used
- **SQL:** For data cleaning and aggregation.
- **Tableau:** For interactive data visualization. https://public.tableau.com/views/StrategicAnalysisofNFCAdoptionPotentialinBakuMetroanditsCorrelationwithBankingSectorTrends/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

## 📈 Key Insights
- **Top Stations:** Koroglu, 28 May, and 20 Yanvar are priority hubs for digital payments.
- **Growth:** Observed a 22.1% growth rate in credit volume as of 2024.
